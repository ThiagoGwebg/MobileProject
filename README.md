# 📱 MobileProject

This repository contains a mobile application project developed for the **Mobile Systems** subject of the **Technical Course in Internet Informatics at ETEC**.

The purpose of this repository is to store exercises, experiments, and practical activities developed during the classes, focusing on learning **mobile application development using React Native and Expo**.

---

## 🚀 Technologies Used

The following technologies are used in this project:

* **React Native** – Mobile application development
* **Expo** – Tool for running and testing the application
* **JavaScript / TypeScript** – Programming languages used in the project
* **Node.js** – Dependency management
* **Git & GitHub** – Version control

MobileProject

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

git clone https://github.com/ThiagoGwebg/MobileProject.git

---

### 2️⃣ Navigate to the project folder

cd MobileProject

---

### 3️⃣ Install dependencies

npm install

---

### 4️⃣ Start the project

npx expo start

---

After starting the project, you can run the application using:

* 📱 **Expo Go** on your mobile device
* 🤖 **Android Emulator**
* 🍎 **iOS Simulator**
